Tutorial Rápido do AvaDisPro 

Este tutorial mostra como utilizar o AvaDisPro através dum passo-a-passo bem simples:

 

1)    Em primeiro lugar, você deve criar um arquivo texto com o seu conjunto de dados. Os dados podem ser de dois tipos: brutos ou tabulados. Nesse exemplo, utilizaremos o arquivo que acompanha o AvaDisPro (Dados\Agrupados2.txt), que tem o seguinte formato:

 

| AvaDisPro - Arquivo de Exemplo 2 (Dados Tabulados)

|

| Os intervalos podem ter amplitudes diferentes.

 

0  25   5

25 35   9

35 45  18

45 55  11

55 70   7

 

| Por Claudio Kinzel, 15/7/2003

 

2)    Utilizar a opção Arquivo -> Dados Tabulados e selecionar o arquivo a testar. O AvaDisPro irá mostrar a tela a seguir:


 

3)    Para testar o seu conjunto de dados usando uma distribuição específica, selecione o menu Distribuição -> Contínua (no caso dos dados serem reais) e selecione a distribuição a ser testada. Nesse exemplo, testou-se a distribuição normal (você pode modificar os parâmetros de cada distribuição).
 

 

4)    Clique em Testar. O AvaDisPro automaticamente irá executar os testes Qui-Quadrado e K-S, conforme figura abaixo:
 

 

5)    No espaço Resultado, você pode visualizar se a distribuição foi aceita ou não. Ao clicar em Detalhes, você pode visualizar um relatório completo do teste realizado (este relatório é sempre salvo no diretório do AvaDisPro, sob o nome de arquivo output.txt).
 

 

6)    Você pode repetir este procedimento para cada distribuição, ou então utilizar o teste para todas distribuições. Para aplicar o teste K-S, utilize o menu Testes -> KS e então clique em Detalhes. Nesse caso, três distribuições foram aceitas e você pode visualizar em quais níveis isso ocorreu.

 

 

7)    Pronto! Estes são os procedimentos para utilizar o AvaDisPro. Se você quiser testar automaticamente o seu conjunto de dados, marque a opção Opções -> Executar Automaticamente -> Qui-Quadrado (ou K-S). Assim, quando você abrir um arquivo, o AvaDisPro já fará os testes, bastando você clicar em Detalhes.
