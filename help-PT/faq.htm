FAQ

O que é o AvaDisPro?

É um software que, dado um conjunto de dados quaisquer, permite determinar se tal conjunto de adapta a alguma distribuição de probabilidade através dos testes Qui-Quadrado e K-S.


Quais distribuições são suportadas?

As seguintes distribuições são suportadas pelo AvaDisPro: Exponencial, Log-Logistic, Gama, Normal, Triangular, Uniforme e Weibull, entre as contínuas, e Binomial, Binomial Negativa, Geométrica, Poisson e Uniforme Discreta, entre as discretas.


Como posso avaliar meus dados com o AvaDisPro?

R: O AvaDisPro lê dois tipos de dados: brutos, isto é, sem qualquer tipo de formatação, e agrupados, os quais já devem estar divididos em classes pré-determinadas. Para tal, simplesmente crie um arquivo texto ordinário e insira seus dados nele. Depois, basta selecionar a opção apropriada no menu Arquivo. Veja os exemplos contidos no diretório Dados para formatar seus dados no formato do AvaDisPro.


O AvaDisPro é livre (como o GNU/Linux)?

Não, ele é quase livre. Na verdade, o seu uso é livre para fins de ensino, pesquisa ou avaliação. Para uso comercial, no entanto, o AvaDisPro não é livre e não pode ser utilizado. Veja o item Licença de Uso para mais detalhes.


O que significa AvaDisPro?

Significa: Avaliador de Distribuições de Probabilidade.


O que é o CP-AvaDisPro?

É um módulo auxiliar que retorna a FDP e a FPA de cada uma das distribuições implementadas pelo AvaDisPro.

Qual a sua motivação para criar esse software?

Na verdade, sempre tive a idéia de fazer alguma coisa interessante em meu trabalho de conclusão. E desenvolver uma ferramenta como essa, que une duas áreas das quais tenho interesse, foi gratificante para mim. Outra questão diz respeito à escassez de softwares de apóio ao processo de aprendizagem estatístico. Mais precisamente na área de distribuições, onde se precisa fazer muitos passos manuais (ou semi-manuais com auxílio de alguma planilha eletrônica), este tipo de software cai como uma luva.
