Bem-vindo à ajuda do AvaDisPro (Avaliador de Distribuições de Probabilidade).

--> Para um início rápido, sugiro ler o Tutorial do AvaDisPro, que contém os passos básicos para a execução do software.

--> Para ler algumas respostas às perguntas freqüentes, selecione a opção FAQ.

--> Para aprender mais sobre os conceitos básicos empregados no AvaDisPro, selecione a opção de seu interesse no menu à esquerda, grupo Conceitos Básicos.

--> Para aprender mais sobre distribuições discretas, selecione os links do grupo Distribuições Contínuas.

--> Para aprender mais sobre distribuições discretas, selecione os links do grupo Distribuições Discretas.

--> Para aprender mais sobre os testes de adequação Qui-Quadrado e K-S, selecione os links do grupo Testes de Adequação.

--> Finalmente, se quiser consultar a bibliografia utilizada, acesse o link Referências.
